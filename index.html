Flask 小筆記_1 - 建立Flask 是甚麼?Flask 是一個使用 Python 編寫的輕量級 Web 應用框架。 如何建立專案?話不多說馬上執行以下步驟 首先打開你的 VSCode，並且開啟你的專案資料夾 新增檔案並命名 (結尾要 .py) 在 app.py 輸入 12345678910from flask import Flaskapp = Flask(**name**)@app.route("/") #裝飾器 @app.route("url")def home():return "Hello World"if **name** == "**main**":app.run() 執行你的 Python $ python app.py 看到這個畫面的話代表你網頁建置好了 開啟瀏覽器輸入網址http://127.0.0.1:5000 就大功告成囉
