錯誤註冊

狀態碼

Flask 中發生錯誤時，會返回一個 HTTP 狀態碼。狀態碼 400-499 表示客戶端的請求數據或者與之相關的錯誤，500-599 表示服務器或應用本身的錯誤。

詳細 HTTP 碼資訊

狀態碼 : 404 Not found

代表著服務器找不到請求資源，在瀏覽器中意味著無法識別 URL，在 API 中意味著端點有效但資源本身不存在。
