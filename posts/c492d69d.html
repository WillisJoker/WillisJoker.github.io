Git 是甚麼?


Git 是一個分散式版本控制軟體，它可以幫助你在做開發時能夠幫助記錄你的所有修改版本，並且讓你能隨時開啟特定版本的系統。

Git 檔案傳送的四個階段

  • Working Directory (工作目錄)

    檔案變動及更改的區域

    檔案狀態 : Untracked files(未追蹤)、Changes not staged for commit(已更改)

  • Staging Area (暫存區)

    將變動及更改的檔案暫時存取的區域

    檔案狀態 :Changes to be committed(等待提交)

  • Local Repository (本地存儲庫)

    本地存儲檔案的倉庫

    檔案狀態 : Committed(已提交)

  • Remote Repository (遠端存儲庫)

    遠端存儲檔案的倉庫

    檔案狀態 : Committed(已提交)

Git 安裝

  • Git 官網

  • 進去點選 Downloads

  • 選取你的作業系統，我是使用 Windows

  • Windows 使用者需要選取你的 Bit

  • 點選 Next

  • 選取你的安裝路徑，在按下 Next

  • 後面就是一些功能或者細項，基本上按 Next 就 OK 了，到這個畫面時按下Install 就可以等它安裝完成啦。
