簡略說明 HDFS

HDFS 是 Hadoop 分佈式文件系統（Hadoop Distributed File System）的縮寫。它是一種用於存儲和處理大型數據集的分佈式文件系統。

查看版本

hdfs dfs version

建立 HDFS 目錄

hdfs dfs -mkdir /hdfs path

列出目錄文件

hdfs dfs -ls /hdfs path

上傳 HDFS 目錄文件

hdfs dfs -put /loaction path /hdfs path

下載 HDFS 目錄文件

hdfs dfs -get /hdfs path /loaction path

刪除 HDFS 目錄文件

hdfs dfs -rm /hdfs path

查看 HDFS 文件內容

hdfs dfs -cat /hdfs path

查看 HDFS 幫助文件

hadoop fs -help
