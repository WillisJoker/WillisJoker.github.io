.NET 學習

Development Enviroment

Visual Studio

  • 官方整合開發環境 (IDE)。
  • 專為 .NET 開發而設計。
  • 提供程式碼編輯器、除錯工具、GUI 設計工具等功能。

Visual Studio Code

  • 輕量級跨平台程式碼編輯器。
  • 支援多種語言和框架，包括 .NET。
  • 擴充功能豐富，可根據需求自定義開發環境。

JetBrains Rider

  • 跨平台 .NET 和 C# 開發工具。
  • 提供程式碼編輯、除錯、重構和整合版本控制等功能。
  • 非免費軟體。

What is .NET ?

Introduction

.NET 是免費的跨平台開放原始碼開發人員平台，可用來建置多種應用程式，它提供了一系列工具、技術和語言，使開發人員能夠快速地創建可靠、安全且高效的軟體。

Programming Languages

  • C#

    .NET 的主要程式語言之一，由微軟開發。它是一種現代化、型別安全的物件導向程式語言。

  • F#

    一種功能強大的函數式程式語言，它具有強大的並行和異步程式設計支援，適用於處理大規模數據和並行計算等場景。

  • VB .NET

    Visual Basic 的進化版本，雖然現在發展語言水平較低，但還是有人會使用。

.NET Architecture

  • Common Intermediate Language (CIL)

    通用中間語言，一種屬於.NET 框架的低階且人類可讀的程式語言。

  • Common Language Runtime（CLR）

    CLR 負責管理.NET 應用程式的執行。

    • 主要功能

      基礎類別庫支援 Base Class Library Support、記憶體管理 Memory Management、執行緒管理 Thread Management、垃圾回收 Garbage Collection、安全性 Security、類型檢查 Type Checker、異常管理 Exception Manager、除錯管理 Debug Engine、中間碼(MSIL)到機器碼(Native)編譯、類別裝載 Class Loader

  • Visual overview
