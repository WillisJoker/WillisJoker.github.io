Revit API 介紹

Revit API 是 Autodesk 提供的工具和功能，旨在讓開發人員以程式化方式擴展和自動化 Revit 建築信息建模軟件，從而控制元素、操作和數據，滿足建築、結構和設備等工程領域的需求。

ADDIN (擴充功能)

管理 Visual Studio 工具擴充功能的 XML 檔案，提供了有關這些擴充功能的元資料資訊，以便 Visual Studio 能夠正確載入和管理它們。

  • ADDIN 文件介紹

DLL (動態連結庫)

DLL 檔案是一種程式庫，它可以被多個程式共享，用來提供各種功能，如圖形界面、網絡通訊、數據庫訪問等。

接口方式

  • IExternalCommand
  • IExternalApplication

實作

Version

Revit : 2021 > Revit SDK : 2021 > .NET Framework : 4.8

需要安裝套件

  • Revit 下載
  • Revit SDK 下載 (最近三個版本)
  • Revit 2021.1 SDK 下載
  • Visual Studio 下載

Revit API 的兩種接口方式

IExternalCommand

當用戶在 Revit 中執行你的外部命令時，Revit 會創建一個 ExternalCommand 的實例，並調用其 Execute 方法。

  • 首先選取 Revit SDK 中的 Autodesk.AddInManager.addin

  • 丟到 Autodesk 中你使用 Revit 版本的資料夾裡

  • 打開 Autodesk.AddInManager.addin 修改 的內容

    “C:\ProgramData\Autodesk\Revit\Addins\2021\Autodesk.AddInManager.Command.dll”

  • 開啟 Visual Studio 建立新專案，選擇類別庫(.NET Franework)

  • 命名專案名稱與選擇架構後建立

  • 打開專案後，去方案總管對參考點右鍵後選擇加入參考

  • 點擊瀏覽

  • 選擇”RevitAPIUI.dll” “RevitAPI.dll” 後按加入

  • 加入成功後按確定

  • 另外從組件找到 System.Windows.Forms 加入後確定

  • 將程式碼貼進來

  • 按下開始

  • 偵錯成功後會出現這個畫面

  • 接著打開 Revit 的模型檔

  • 選增益集的外部工具後，按 Add in Manager

  • 選擇 Load

  • 找到你創建的 dll 檔案

  • 按下 Run

  • 你的程式碼就可以成功執行了

IExternalApplication

當 Revit 啟動時，會初始化 ExternalApplication 的實例，並在特定的時機點調用其方法。

  • 在你使用 Revit 版本的資料夾裡新增一個記事本

  • 將以下的定義寫入記事本

  • 另存新檔為 .addin

  • 接著將程式碼貼進來

  • 按下開始

  • 偵錯成功後會出現這個畫面
