2022-鐵人賽 後端幼幼班Day22 MySQL篇 資料庫基礎操作

大家好，我是 Willis，今天要來介紹 MySQL 資料庫基礎的操作喔 ! 加油 ! (⌐▀͡ ̯ʖ▀)

資料庫操作

創建資料庫 - Create database

  • 打開 MySQL 的 Workbench，這裡就是你寫下 MySQL 指令的地方

  • 寫下創建資料庫的指令

  • 選取指令後按下閃電的按鈕執行它，可以看到創建成功的訊息

讀取資料庫 - Read database

  • 執行指令後，可以看到所有的資料庫，包括你剛剛創建的 sql_test

刪除資料庫 - Delete database

  • 試著把剛剛創建的資料庫刪除吧

  • 讀取一下資料庫看看，發現 sql_test 被刪除了

使用資料庫 - Use database

  • 在創建資料表格前先把剛剛刪除的資料庫創建回來吧 XD

  • 輸入使用資料庫的指令後，你就可以使用該資料庫來做你的任務啦
