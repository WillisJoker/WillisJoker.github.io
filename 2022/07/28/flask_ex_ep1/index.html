Flask 小筆記_番外 1 - JSON

JSON 是甚麼?

JSON 是獨立於程式語言的資料格式，其不僅是 JavaScript 的子集，也採用了 C 語言家族的習慣用法，目前也有許多程式語言都能夠將其解析和字串化。

與 Flask 的關係

可做為 Flask 傳遞資料的方法。

推薦小工具-Postman

  • Postman

    Postman 是一個 API 平台，供開發人員設計，構建，測試和迭代其 API。

判斷 JSON

request.is_json

  • 程式碼 app.py

Postamn 使用方法

  • 打開 Postman 並且新增一個操作頁面

  • 按下 Send 察看結果，得知 JSON 檔傳送成功
