前言

大家好~我是 Willis 是個大學生，今天是鐵人賽第一天，這 30 天我會把我學習到的知識及經驗分享給大家，多多指教啦。
加油! (//●⁰౪⁰●)//

Python 是甚麼?

一種廣泛使用的高階和通用的程式語言，他強調代碼的可讀性與簡潔姓，相比於 C 或 Java，Python 能讓開發者用更少的代碼表達想法。

Python 安裝

  • Python 官網

  • 進去點選 Downloads 找到你的作業系統，我是 Windows 所以直接按 Python 3.10.6

  • 點選 Install Now

  • 接著它就會告訴你安裝成功囉!

Visual Studio Code 安裝

  • Visual Studio Code 官網

  • 進去點選 Download Windows

  • 點 下一步 到最後，看到此畫面點選 安裝

  • 打開你的 VS Code 點選 延伸模組 (四格方塊的 icon)，搜尋 python 後點選 安裝

第一支 Python 程式

終於把環境都建置好了，接下來我們就來寫 Python 的第一支程式吧。

  • 打開你的 cmd 命令工具 (推薦 Cmder)，檢查一下你的 Python 版本吧。
  • 找一個儲存位置(我選擇在 E 槽)，建置你的專案資料夾(我取名叫 : my_project)
  • 移動至你的專案資料夾
  • 開啟 VS Code

  • 新增一個檔案，這時候就可以寫下你的程式啦!

  • 程式碼 app.py

  • 打開 cmd 執行一下
